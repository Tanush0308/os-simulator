#  OS Simulator (Web-Based)

A full-stack interactive Operating System simulator built using **React + Node.js**.

---

##  Features

###  CPU Scheduling

* FCFS
* SJF
* Round Robin
* Priority Scheduling
* Gantt Chart Visualization
* Process Table (AT, BT, CT, TAT, WT)
* Live Simulation (Ready → Running → Completed)

###  Memory Management

* First Fit
* Best Fit
* Worst Fit
* Fragmentation Visualization

###  Deadlock Detection

* Resource Allocation Graph
* Cycle Detection

###  Disk Scheduling

* FCFS
* SSTF
* SCAN
* C-SCAN

---

##  Tech Stack

* Frontend: React + Tailwind CSS + Framer Motion
* Backend: Node.js + Express
* Execution Engine: Python (via child_process)

---

##  Project Structure

frontend/ → UI + animations
backend/ → APIs + algorithms + Python execution

---

##  How to Run

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
node server.js
```
